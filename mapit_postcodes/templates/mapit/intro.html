MapIt is a service that maps geographical points to administrative areas. It’s useful for anyone who has the co-ordinates of a particular point, and needs to find out what regions it lies within. It’s also great for looking up the shapes of all those boundaries.

This particular instance of MapIt contains approximate boundaries for postcodes, postcode sectors, postcode districts and postcode areas, based on the Voronoi diagram of postcode centroids from ONSPD. You can also read more about the project to make these boundaries.

You can download from GitHub the the source for the core Django MapIt application, or the source code specific to this instance (i.e. everything to do with approximate postcode boundaries).
